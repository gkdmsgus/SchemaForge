사용 방법
① 구체적으로 입력

부품 모델명까지 명시하면 정확도가 높아져요.
❌ "앰프 회로"
✅ "LM386 베이스 앰프 회로"

② 용도를 설명

전압, 전류, 목적을 함께 쓰면 더 완성도 높은 회로가 나와요.
✅ "5V USB 입력, 스피커 출력 앰프"

③ 다운로드 & KiCad

.net 파일 다운로드 후
KiCad → File → Import → Netlist
로 스키매틱에 불러올 수 있어요.

④ 추천 입력 패턴

[IC 모델명] + [용도] + 회로
예: "TP4056 리튬배터리 충전 회로"
예: "NE555 LED 점멸 회로 1Hz"
